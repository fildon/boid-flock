# Boids Canvas

A React component to render a boids simulation anywhere

## Installation

```shell
npm add 'boid-flock'
yarn add 'boid-flock'
```

## Usage

```ts
import { BoidFlock } from "boid-flock";

export function YourComponent() {
  return (
    <BoidFlock>
      <span>You can render whatever you like in here</span>
    </BoidFlock>
  );
}
```
